# Superstore Sales Dashboard

## Project Overview

This project focuses on creating an interactive dashboard using Power BI and the Superstore Dataset. The dashboard provides insights into sales, profit, and regional performance through visualizations and interactive filters.

## Tools Used

- Power BI Desktop

## Dataset

- Sample Superstore Dataset

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit

### Visualizations
- Sales by Category (Bar Chart)
- Profit by Region (Pie Chart)
- Sales Trend by Order Date (Line Chart)

### Interactive Slicers
- Category
- Region
- Segment

## Objectives

- Analyze sales and profit performance.
- Compare sales across product categories.
- Identify regional profit distribution.
- Monitor sales trends over time.
- Enable interactive filtering through slicers.

## Deliverables

- Power BI Dashboard File (.pbix)
- Dashboard Screenshots

## Key Insights

- Technology category generated the highest sales.
- Regional profit contribution varies across locations.
- Sales trends can be analyzed over different time periods.
- Interactive filters allow dynamic data exploration.

## Outcome

This project helped in understanding dashboard creation, data visualization, interactive reporting, and business intelligence concepts using Power BI.
